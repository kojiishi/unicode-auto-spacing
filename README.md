# Unicode-CJK

## Installing

This package requires [PyICU], which requires some libraries installed.
Please see the [Instaling PyICU](https://gitlab.pyicu.org/main/pyicu#installing-pyicu).

[PyICU]: https://pyicu.org

```shell-session
pipenv install
```

## Build the Data File

```shell-session
./scripts/build.sh

```
